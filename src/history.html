Projeto Final

Neste trabalho criamos três visualizações para responder três questões de pesquisa, que utilizam dados de dois datasets. O primeiro conjunto de dados compreende os erros de vários sistemas de recomendação para um conjunto de validação do dataset Movie Lens 20M (ML20m). O segundo conjunto de dados compreende uma série de atributos (features) dos usuários do ML20m.

Pergunta 1 - Os Sistemas de Recomendação cometem erros parecidos?

Pergunta 2 - Conseguimos identificar características do usuário que fazem com que algoritmos de recomendação tenham performance ruim?

Pergunta 3 - Conseguimos identificar grupos de usuários com características parecidas e que possuem performance ruim nos sistemas de recomendação?
